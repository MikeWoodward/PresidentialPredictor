silkworm

Image source: White House | License: Creative Commons

silkworm forecasts the results of US presidential elections using the previous election results and current polling data. It gets its name from the fact that US President John Quincy Adams 'kept' silkworms (see Presidential Pet Museum). 2020 data is sourced from FiveThirtyEight.com, 2016 data and before is sourced from the Pollster API. You'll need an API key for the Pollster data if you want to retrieve 2016 and before data.

I designed silkworm to be relatively straightforward to use, there's help on each panel and the project home page on Github has more help.
